# Smart-Knob
Features/devices:
* haptic motor    - NodeMCU managed
* angle sensor    - NodeMCU managed
* display         - NodeMCU managed
* strain gauge    - PIC managed
* ring leds       - PIC managed
* light sensors/display brightness - PIC managed
